Platform Design Madness

What is Platform Design Madness?
Short and simple, Platform Design Madness (PDM) is an editor for two-dimensional platform stages.
Users can easily create 2D game levels by specifying a start point, end point, and environment blocks.
Stages can be saved for later (to be edited or played), as well as be upvoted by other users.
